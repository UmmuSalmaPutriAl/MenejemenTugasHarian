# Software Design Document (SDD)

## 1. Arsitektur Sistem
Model: Client-side web app (HTML, CSS, JavaScript, LocalStorage)

## 2. Desain Database
Entity: Task
- id (int)
- title (string)
- description (string)
- status (boolean)

## 3. Desain UI
- Halaman utama: daftar tugas
- Tombol: Tambah, Edit, Hapus, Tandai selesai
- Filter: Semua, Selesai, Belum
